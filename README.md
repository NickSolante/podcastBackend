# podcastBackend

## Getting Started

This project is built using [express.js](https://expressjs.com/).

The following steps will get you setup to run a local development version of the podcastBackend.

### Prerequisites

- [Node.js](https://nodejs.org/en/): JavaScript run-time and main development language.
- [npm](https://www.npmjs.com/get-npm): Node.js package manager.

### Setup

```bash
git clone https://github.com/
cd podcastBackend

# init backing services
./services_up.sh
```
